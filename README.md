
# 2024/7/8 更新 V1版本
微信小程序用户地址获取（wx.getLocation），拿到经纬度后调用腾讯的逆地址解析服务需要收费！！由此开发通过经纬度转化成行政区划，适合不需要特别精确的定位场景。

输入经纬度数据转换成地址--输出中国的行政区划省、市、区名称及其区域代码
